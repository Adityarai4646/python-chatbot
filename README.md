Technologies Used:

Features:

- Responds to greetings like `"hi"`, `"hello"`, `"hey"`
- Answers simple questions such as `"how are you?"`
- Handles conversation ending with `"bye"`, `"exit"`, or `"quit"`
- Includes fallback responses for unknown queries
